This repository contains components developed as part of **Work Package 3 (WP3)** of the SustainML project.

WP3 focuses on assessing and reporting the **sustainability impact of machine learning workflows**, including energy consumption estimation, carbon footprint analysis and runtime monitoring and sustainability-related metrics.

## Scope

The components in this repository are designed to:

- Collect execution and resource usage information
- Estimate energy usage and emissions for training and inference
- Expose sustainability-related metrics to the SustainML Framework

## Usage

WP3 module is not intended to operate independently and are integrated into the broader SustainML pipeline.

For installation, configuration, and execution instructions, refer to the main framework repository:

https://github.com/eProsima/SustainML-Framework
